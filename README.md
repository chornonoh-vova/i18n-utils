# i18n-utils

Collection of i18n utilities:

- `add` - adds a specified key/value to the translation files
- `update` - updates a specified key with new value in translation files
- `remove` - removes a specified key from the translation files

```bash
i18n-utils --help
```

## Requirements

- Node.js v24+
- npm v11+

## Development

Installing dependencies:

```bash
npm install
```

Building:

```bash
npm run build
```

Linking the package locally:

```bash
npm run link
```

Running with debug logs

```bash
DEBUG=* npm start
```

## Releasing

```bash
npm run release
```
